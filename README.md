## ✨ What is CopyTXT?

CopyTXT is a **free all-in-one notepad** and multi-format editor for Windows 10 and 11.
Instead of juggling a text editor, a spreadsheet program and a PDF viewer, you open and
manage **every kind of text and document in a single, tabbed window** — fast, private,
and easy to use.

It's built to feel as quick as the built-in Windows notepad, while doing far more.

## 🚀 Features

- **All-in-one formats** — open and edit plain text, code, Markdown, CSV/TSV, Excel
  spreadsheets, JSON and config files, and view PDFs — side by side in tabs.
- **Fast & autosaving** — opens almost instantly, saves as you type, and restores your full
  session (open tabs + window position) the next time you launch it.
- **Secure & private** — your files stay on your computer and the app works fully offline.
  An optional, encrypted Google Drive backup is there only if *you* enable it.
- **A tidy file vault** — organize notes into infinitely nested groups with instant search
  and drag-and-drop.
- **Spreadsheet grid** — a Sheets-style editor for CSV/TSV/XLSX with formulas and colors.
- **Markdown, done right** — clean rendering with per-block "copy" buttons, full-height
  editor and find/replace.
- **Comfort controls** — adjustable font, text size and brightness, per file.
- **Light & dark themes** — a warm light theme by default, deep dark mode one tap away.

## 📥 Installation

1. Go to the [**Releases**](../../releases/latest) page.
2. Download **`CopyTXT-Setup.exe`**.
3. Run the installer (Windows 10 / 11, 64-bit) and follow the prompts.
4. Launch CopyTXT and start writing — **100% free**, no account, no setup.

### "Windows protected your PC" warning?

CopyTXT is a small independent app and isn't code-signed yet, so Windows SmartScreen may
show a blue warning the first time you run the installer. This is expected. To continue:

> Click **More info** → **Run anyway**.

The download is safe — you can always re-download it from the official
[Releases page](../../releases/latest) or from [copytxt.com](https://copytxt.com).

## 💻 System requirements

- Windows 10 or Windows 11 (64-bit)
- Microsoft Edge **WebView2 Runtime** (already installed on most up-to-date Windows systems;
  the installer will guide you if it's missing)

## 🔒 Privacy

CopyTXT stores everything **locally** on your machine. It does not phone home or upload your
files. The only network feature is an **optional** Google Drive backup that you turn on
yourself.
